Introduction

vstat is a C++20 library of computationally efficient methods for calculating sample statistics (mean, variance, covariance, correlation), common regression metrics ( \(R^2\) score, MSE, MAE) and likelihoods (Gaussian and Poisson).

The library has the following features:

  • the implementation builds upon the SIMD abstraction layer provided by E.V.E (The Expressive Vector Engine)
  • it uses a data-parallel version of the numerically-stable algorithm from Edward A. Youngs and Elliot M. Cramer where the results from independent data partitions are combined with the approach by Schubert and Gertz
